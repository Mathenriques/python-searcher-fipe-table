# Buscador de Carros na tabela FIPE

Esse projeto consiste na busca de carros de forma aleatoria na tabela FIPE, retornando todas as informações, como:

- Mês de referência;
- Código Fipe;
- Marca;
- Modelo;
- Ano Modelo

## Pré Requisitos
- Python >= v3.10
- Pip  v22.0
- Selenium >= v3.141.0

## Inicializar

Abra o shell na pasta do projeto e execute:

```shell
  python main.py
```
